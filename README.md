# Mission Finder

A [Guild Wars 2](https://www.guildwars2.com/) addon for the [Raidcore Nexus](https://raidcore.gg/Nexus) loader. It shows a searchable, tabbed in-game window of guild mission locations. Click a row to copy the waypoint to your clipboard or send it straight to chat.

This repository hosts the **releases and downloads**. Grab the latest `MissionFinder.dll` from the [Releases](../../releases) page, or let Nexus update it automatically.

## ⚠️ Notice

A future release will read Guild Wars 2's memory to detect which guild missions are currently active and which targets you've already completed. If you'd rather not have an addon read game memory, uninstall Mission Finder or disable auto-updates before that release. The addon shows a one-time in-game notice ahead of the change.
